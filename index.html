    • Internet

    A Internet é uma rede de redes que foi desenvolvida no fim dos anos 60 pelo Departamento de Defesa dos EUA, com a finalidade de criar uma rede de comunicação descentralizada que pudesse conter um ataque nuclear.
    A Internet funciona através de dispositivos e sistemas de computadores que juntos usam um conjunto de protocolos padronizados.
    Um protocolo define como informações são trocadas entre os dispositivos e garante que os dados sejam transmitidos de apropriado e com segurança.

  • Packet: É uma pequena unidade de dados que é transmitido dentro da Internet.
  • IP Address: É um identificador único associado a cada dispositivo em uma rede, usado para rotear dados para o destinatário certo.
  • DNS: O Nome do Domínio do Sistema (Domain Name System) é responsável por traduzir nomes de domínio em Endereços de IP.
  • HTTPS: É uma versão encriptografada de HTTP que é usado para providenciar uma comunicação segura entre o cliente e o servidor.
  • Router: Um dispositivo que direciona pacotes de dados entre diferentes redes.
  • Domain Name: Um nome legível que é usado para identificar um website, tanto quanto google.com.
  • HTTP: O Hypertext Transfer Protocol é usado para transferir dados entre o cliente (quanto a um navegador da web) e o servidor (quanto a um website).
  • SSL/TLS: The Secure Socket Layer e Transport Layer Security são protocolos usados para providenciar uma comunicação segura dentro da Internet.
